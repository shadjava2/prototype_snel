Pour compléter la PWA, vous devez créer deux icônes PNG :
- icon-192.png (192x192 pixels)
- icon-512.png (512x512 pixels)

Vous pouvez utiliser n'importe quel logo ou image représentant le Ministère du Travail RDC.
Pour un prototype rapide, vous pouvez utiliser un générateur d'icônes en ligne ou créer des images simples avec les couleurs RDC (#0051A8, #FCD116, #CE1126).

Ces icônes sont référencées dans public/manifest.json.
